# Files

- NB_DE_dict.jpeg: cover image (useless, but required for creating the MOBI file)


```
kindlegen.exe NB_DE_dict.opf -c2 -verbose -dont_append_source
```

Copy the MOBI file to `documents/dictionaries/` on your Kindle.
